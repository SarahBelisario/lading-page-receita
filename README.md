# 🧁 Landing Page - Cupcake de Café com Chantilly

Uma página web moderna e responsiva desenvolvida para apresentar uma receita detalhada de Cupcake de café com chantilly, com foco em uma interface limpa, legível e visualmente atraente.

## 🚀 Sobre o Projeto

Este projeto foi criado como parte dos meus estudos de desenvolvimento web (Aulas Rocketseat). O objetivo principal foi estruturar o escopo básico de uma página de receitas utilizando boas práticas de HTML e CSS, focando na organização do conteúdo e na estilização de elementos como listas, imagens e cabeçalhos.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica de todo o conteúdo da receita.
* **CSS3:** Para a estilização visual, incluindo cores personalizadas, tipografia e espaçamentos.
* **Git & GitHub:** Para o controle de versão e hospedagem do código.

## 📝 Funcionalidades da Página

* **Seção Principal:** Apresentação do prato com título descritivo e uma imagem de alta qualidade.
* **Informações Rápidas:** Exibição clara do tempo de preparo (1h10), rendimento (12 porções) e nível de dificuldade (fácil).
* **Lista de Ingredientes:** Organização limpa e escaneável utilizando listas estruturadas.
* **Modo de Preparo:** Passo a passo detalhado para a execução da receita (como a finalização com o chantilly reservado e os grãos de café).

## 👩‍💻 Como Executar o Projeto Localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd lading-page-receita
   ```
3. Abra o arquivo `index.html` em qualquer navegador web.
